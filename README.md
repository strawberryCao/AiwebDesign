# Polar Index

An original, procedural WebGL spatial website study inspired by the interaction quality of leading immersive studios. It uses Three.js, GSAP, Vite and Cloudflare Workers static assets.

## Local
```bash
npm install
npm run dev
```

## Deploy
Add GitHub Actions repository secrets `CLOUDFLARE_API_TOKEN` and `CLOUDFLARE_ACCOUNT_ID`, then push to `main`. The workflow builds and deploys through Wrangler.
